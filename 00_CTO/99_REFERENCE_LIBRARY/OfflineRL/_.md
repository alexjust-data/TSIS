# CAPA 1 — LOS PAPERS FUNDACIONALES (OBLIGATORIOS)

Estos son los que estudiarían los ingenieros de DeepMind.

## 1. AlphaGo (2016)

Mastering the Game of Go with Deep Neural Networks and Tree Search

Qué aprender:

```text
Policy Network
Value Network
Monte Carlo Tree Search
Self-play
```

Es el nacimiento de AlphaGo.

No trading.

No LLM.

Solo la arquitectura original.

([Wikipedia][1])

## 2. AlphaGo Zero (2017)

Mastering the Game of Go without Human Knowledge

Aquí ocurre la revolución.

DeepMind elimina:

```text
human expert games
```

y deja:

```text
rules
+
self-play
```

AlphaGo Zero supera a AlphaGo aprendiendo desde cero. ([Google DeepMind][2])

## 3. AlphaZero (2017)

Mastering Chess and Shogi by Self-Play with a General Reinforcement Learning Algorithm

Este es probablemente el paper más importante para TSIS.

¿Por qué?

Porque DeepMind demuestra:

```text
mismo algoritmo
↓
Go
Chess
Shogi
```

sin rediseñar la solución para cada juego. ([arXiv][3])

## 4. MuZero (2019)

Mastering Atari, Go, Chess and Shogi by Planning with a Learned Model

Este es el paper que más se parece a tu problema.

MuZero:

```text
NO conoce reglas
NO conoce dinámica
```

Eso es exactamente lo que un mercado requiere.
