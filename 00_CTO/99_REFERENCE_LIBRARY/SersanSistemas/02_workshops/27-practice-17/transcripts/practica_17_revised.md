**¿Qué es este código?**

Un sistema de trading que usa datos COT (Commitment of Traders) para detectar qué están haciendo los grandes especuladores del mercado y seguir su dirección.

**El indicador clave:** Calcula la *posición neta* de los Non-Commercials (fondos de inversión, hedge funds):
- Posición neta = Contratos largos − Contratos cortos
- Luego lo convierte a escala 0-100 comparando con su rango histórico de 55 períodos

| Compra (Long) | Vende (Short) |
|---------------|---------------|
| Los grandes están netos largos | Los grandes están netos cortos |
| El índice está aumentando | El índice está disminuyendo |
| Índice en zona media (10-90) | Índice en zona media (10-90) |

**Gestión del riesgo:**

- Stop loss dinámico en el mínimo de las últimas 3 barras
- Salida si el índice alcanza extremos o cambia el signo de la posición neta

```sh

inputs:
	Length (55),
	C_Trigger (10),
	Cortos (True),
	Per_Out (3);

variables:
	S_Trigger (90),
	COTIndexNonComm (0),
	NonCommNet (0),
	NonCommHighest (0),
	NonCommLowest (0),
	Stop_Long (0);

S_Trigger = 100 - C_Trigger;
Stop_Long = Lowest (Low, Per_Out)[1];

NonCommNet = Getfunddata("COTF-9", 0) -  Getfunddata("COTF-10", 0);
NonCommHighest = Highest (NonCommNet, Length);
NonCommLowest = Lowest  (NonCommNet,  Length);

if currentbar > 1 then
	COTIndexNonComm  =  100 * (NonCommNet - NonCommLowest) / IFF ((NonCommHighest - NonCommLowest) <> 0, NonCommHighest - NonCommLowest, 1);

If COTIndexNonComm > C_Trigger and COTIndexNonComm < S_Trigger and NonCommNet > 0
	and COTIndexNonComm > COTIndexNonComm[1] then
		Buy next bar at market;
		
If Marketposition = 1 Then
Begin
	If Barssinceentry = 0 Then
		Stop_Long = Lowest (Low, Per_Out);
	
	If COTIndexNonComm crosses below S_Trigger or COTIndexNonComm < C_Trigger or  NonCommNet < 0 then
		Sell next bar at market;
	
	sell next bar at Stop_Long stop;
End;

If cortos then
Begin
	If COTIndexNonComm < S_Trigger and COTIndexNonComm > C_Trigger and NonCommNet < 0
	and COTIndexNonComm < COTIndexNonComm[1] then
		Sell Short next bar at market;
		
	If COTIndexNonComm crosses above C_Trigger or COTIndexNonComm > S_Trigger or  NonCommNet > 0 then
```

2. Que el índice esté por debajo de `S_Trigger` (90) — para no comprar en zona de sobrecompra
3. Que `COTIndexNonComm > COTIndexNonComm[1]` — confirmando que el índice está en tendencia ascendente

Condiciones de ***salida LARGA***:

- Cuando el índice cruza por debajo de `S_Trigger` (90) — saliendo de sobrecompra
- O cuando cae por debajo de `C_Trigger` (10) — como seguridad si no llegó a cruzar
- O cuando `NonCommNet` se vuelve negativo
- Stop de protección en `Lowest(Low, Per_Out)` — mínimo de las últimas 3 barras

Operativa ***CORTA***: Lógica inversa: entra corto cuando el índice baja, el saldo neto es negativo, y sale cuando se dan las condiciones opuestas.
