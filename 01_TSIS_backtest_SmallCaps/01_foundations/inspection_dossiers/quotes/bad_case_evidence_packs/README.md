# Quotes Bad Case Evidence Packs

Esta carpeta debe contener los dossiers `bad` de `quotes`.

Unidad esperada:

- un documento agregado del bloque;
- o packs por caso cuando el cierre lo requiera.

Cada caso debe combinar:

- metadatos del `ticker-date-file`;
- analisis forense;
- imagen incrustada;
- decision institucional;
- y trazabilidad al asset original.
