# Quotes Good Justification

Esta carpeta debe contener la justificacion positiva de la franja `good` de `quotes`.

No pretende volcar todo el universo bueno.

Debe contener:

- ejemplos representativos;
- explicacion de por que el crossed leve o el rollover limpio no invalidan el libro;
- y evidencia suficiente para que un inspector entienda por que el caso sigue autorizado.
